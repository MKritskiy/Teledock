# Teledock EN
The project was created using the following technologies: ASP.NET Core 8.0, Entity Framework Core 8, PostgreSQL, xUnit, Swagger, Docker.


To start, open the console in the root folder of the project and enter the command:
```docker
docker-compose up -d
```
After that, two containers will be created orchestrated together in the docker  

At the address http://localhost:12345/index.html The Swagger interface is located.

At the address http://localhost:12345/swagger/v1/swagger.json is the json generated by Swagger

# Teledock RU
Проект создан с использованием следующих технологий: ASP.NET Core 8.0, Entity Framework Core 8, PostgreSQL, xUnit, Swagger, Docker.



Для запуска необходимо открыть консоль в корневой папке проекта и ввести команду:
```docker
docker-compose up -d
```
После этого будет создано два оркестрированных вместе контейнера в докере  

По адресу http://localhost:12345/index.html расположен интерфейс Swagger.

По адресу http://localhost:12345/swagger/v1/swagger.json расположен json сгенерированный посредством Swagger
